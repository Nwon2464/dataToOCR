# v1.2 Text Study Exporter MVP

## Goal

Convert MinerU API output into a content-first study format that is easy to read, select, copy, and reuse for AI-based study.

This version does not try to reproduce the original scanned PDF layout perfectly. Instead, it prioritizes text preservation and study usability.

## Input

A MinerU API output directory such as:

```text
data/mineru_api_output/uscpa_reg2_1_p022_024/
```

Main files used:

```text
*_content_list_v2.json
*_origin.pdf
images/
```

## Script

```text
scripts/export_text_study.py
```

## Example Command

```bash
python scripts/export_text_study.py \
  --mineru-dir data/mineru_api_output/uscpa_reg2_1_p022_024 \
  --output-dir exports/text_study/uscpa_reg2_1_p022_024
```

## Output

```text
exports/text_study/<chunk_name>/
  index.html
  text_study_pageXXX.html
  pageXXX_text_study.md
  normalized_pageXXX.json
  normalized_pages.json
  assets/
```

## Design Policy

The exporter uses a content-first approach.

- Body text is rendered as selectable text.
- Images, charts, tables, and forms are preserved as visual assets when needed.
- Side notes are separated into `補足 / Side Note` blocks when meaningful.
- Page headers, footers, page numbers, and decorative page-aside labels are treated as metadata.
- Original layout fidelity is not the top priority.
- Text selection and copying are prioritized.

## Tested Chunks

The MVP has been tested on:

```text
uscpa_reg2_1_p019_021
uscpa_reg2_1_p022_024
uscpa_reg2_1_p028_030
```

## Findings

The exporter works across multiple page types:

- Text-heavy pages
- Pages with figures
- Pages with tables
- Pages with Form 1040 image content
- Pages with right-side notes

The output HTML is suitable for checking readability and text selection quality.

## Known Limitations

- Complex tables are often preserved as images rather than fully reconstructed as HTML tables.
- Chart blocks may need better dedicated rendering in a future version.
- The current exporter is chunk-based and does not yet batch-process all MinerU output folders.
- PDF export for GoodNotes/PDF apps has not yet been finalized.
- OCR quality still depends on MinerU output quality.
